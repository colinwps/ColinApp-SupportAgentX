# 退款与售后政策

## 退款规则

1. **7天无理由退款**：商品签收后7天内，未使用、未拆封可申请退款，运费由买家承担。
2. **15天质量问题退换**：商品存在质量问题，签收15天内可申请退款或换货，运费由卖家承担。
3. **超期处理**：超过15天的质量投诉，需经客服审核后决定是否受理。
4. **不支持退款情形**：定制商品、数字内容（激活码）、食品类开封后不支持退款。

## 退款流程

1. 用户提交退款申请（需上传凭证）
2. 客服审核（1个工作日内）
3. 审核通过后，退款将在3-5个工作日内原路返还
4. 如需退货，须在获得退款审批后48小时内寄回商品

## 换货政策

- 支持一次免费换货（同款商品，尺码/颜色错误）
- 换货运费由卖家承担
- 换货商品到仓后3个工作日内发出新品
